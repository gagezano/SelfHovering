# SelfHovering

This hub showcases stand-alone experiments crafted from the Arc UTC codebase. Each folder contains a stripped-down version of a feature we found valuable.

## Experiments
- `Image BG Rotation` — A grayscale hero background slideshow with responsive behavior.
- `Simple Jump To Nav` — A sticky, responsive jump navigation example.
- `Custom Image Mapbox` — The map + amenities layout copied directly.
- `Carousel with Sets` — The campus carousel with placeholder "Set" data.
- `Animated Radial Gradient BG Combustion` — The pulsing footer gradient turned into a background.

Open `index.html` to access the experiments via the provided links.
